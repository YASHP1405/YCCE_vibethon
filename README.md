# 🔄 SkillSwap — Agentic Skill Marketplace

> **YCCE VIBE-A-THON 2026 | Problem Statement #2**

AI-powered marketplace where engineering students swap expertise. Our AI agent analyzes GitHub profiles to assign **Skill-Value** scores and matches peers for real-time **1:1 learning sessions**.

## ✨ Features

- 🧠 **AI GitHub Analyzer** — Gemini AI evaluates repos, languages, activity to generate Skill-Value scores
- 🤝 **Smart Matching** — Complementary skill matching engine finds ideal learning partners
- 🏪 **Marketplace** — Browse, filter, and discover peers by skill, city, or college
- 💬 **Live Sessions** — Real-time chat, session timer, and topic tracking
- 📊 **Skill Dashboard** — Radar charts, skill bars, and progress tracking
- 🏆 **Leaderboard** — City-wise rankings with badges & achievements
- ⭐ **Reviews** — Post-session ratings and feedback system

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Python Flask |
| Database | SQLite |
| AI | Google Gemini 2.0 Flash |
| GitHub | GitHub REST API |
| Frontend | HTML5, CSS3 (Glassmorphism), Vanilla JS |

## 🚀 Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Set up environment (copy and edit)
cp .env.example .env
# Add your GEMINI_API_KEY to .env

# Run the app
python app.py
```

Visit `http://localhost:5000` and start swapping skills! 🎉
